# Zakumi

Software y marketing — landing [Next.js](https://nextjs.org) (App Router, GSAP).

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Despliegue

Recomendado: [Vercel](https://vercel.com/new) conectado al repositorio Git.
